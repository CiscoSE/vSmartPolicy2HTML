vSmart Policy to HTML Conversion Tool

Built on the open source code posted at: vSmartPolicy2HTML

This tool will convert the text config of vSmart central policy to an HTML linked page.

NOTE: The tool relies heavily on the proper indention of each line to parse and markup the policy.
If there are indention irregularities in the policy, it may generate errors or incomplete markup.

To convert your policy copy it to a .txt file from vManage:
  • Navigate to the Configuration -> Policies -> Centralized Policy page
  • If using a vManage policy, Preview the active policy
  • If using a CLI policy, View the active policy
  • Copy results and save as a text file with .txt extension
  • Upload the file below
